# GoIT Homework — JavaScript Module 12

## 📘 Опис проєкту

Цей репозиторій містить домашнє завдання з **модуля 12 JavaScript** курсу
**Fullstack Web Developer** від **GoIT**.

Модуль присвячений роботі з **async/await**, обробці помилок, реалізації
**пагінації** та застосуванню підходу **«Load more»** під час взаємодії з REST
API.

У проєкті використовується **Pixabay API** для отримання зображень з підтримкою
пагінації.

---

## 🎯 Результати навчання (Module 12)

Після опрацювання матеріалів модуля ти:

- Розумієш синтаксис **async/await**.
- Вмієш обробляти помилки через `try/catch`.
- Розумієш сутність **пагінації**.
- Вмієш налаштовувати параметри запиту (`page`, `per_page`).
- Реалізуєш функціонал **«Load more» (Завантажити ще)**.
- Контролюєш стан інтерфейсу під час асинхронних операцій.

## 📂 Основний функціонал

### 🔎 Пошук зображень

- Асинхронний запит через `async/await`
- Отримання JSON-відповіді
- Додавання розмітки в DOM

### 📄 Пагінація

- Передача параметрів `page` та `per_page`
- Збереження поточної сторінки
- Перевірка досягнення останньої сторінки

### ➕ Load More

- Кнопка «Load more»
- Додавання нових результатів без очищення попередніх
- Приховування кнопки при завершенні результатів

## 💡 Best Practices

✅ Використовуй `async/await` замість вкладених `.then()`  
✅ Завжди обгортуй асинхронний код у `try/catch`  
✅ Перевіряй `response.ok` перед обробкою відповіді  
✅ Контролюй стан кнопки «Load more»  
✅ Уникай дублювання запитів під час активного завантаження
